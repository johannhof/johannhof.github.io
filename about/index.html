About Me

I work at Mozilla as an engineer on the Firefox Privacy & Security Team. I spend the majority of my time working on our anti-tracking efforts, though I also maintain other areas of browser security and privacy, such as web permissions or data sanitization.

I'm the module owner for Firefox Security and Privacy UI as well as a module peer for a bunch of other modules.

Before working on open source full time at Mozilla I was a pretty active open source contributor, mostly using JavaScript and Rust. If you're interested in my projects you should head over to my GitHub profile.
